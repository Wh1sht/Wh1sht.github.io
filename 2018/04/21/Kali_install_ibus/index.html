Kali Linux中安装Ibus拼音输入法

21 Apr 2018

Ibus是GNU/Linux和UNIX操作系统下GPL协议分发源代码的开源免费语言输入法框架。
Kali Linux中默认是没有安装有中文输入法的。

Start Terminal

Command-line Input

Install an input method

Finish

点击桌面右上角的设置->区域和语言，在输入源中添加汉语(拼音)

Restart

重启电脑，可通过Win+space切换输入法
